# هدف این سند

این سند تمامی تصمیمات UX/UI نهایی محصول «نزدیکام» را مشخص می‌کند و نسبت به تمامی طراحی‌های قبلی اولویت بالاتری دارد.

هر بخشی از رابط کاربری که با این سند مغایرت داشته باشد باید بازطراحی شود.

---

# اصل شماره 1

نزدیکام یک Mobile First Web App است.

بیش از 95٪ کاربران از موبایل وارد خواهند شد.

بنابراین:

* ابتدا موبایل طراحی شود
* سپس همان طراحی برای دسکتاپ توسعه پیدا کند
* ظاهر موبایل و دسکتاپ باید یک هویت بصری یکسان داشته باشند
* نباید دو تجربه کاملاً متفاوت بین موبایل و دسکتاپ وجود داشته باشد

---

# اصل شماره 2

کاربر نباید گیج شود.

هر صفحه باید:

* فقط یک هدف اصلی داشته باشد
* CTA اصلی مشخص باشد
* منوها حداقل پیچیدگی را داشته باشند
* از شلوغی بصری جلوگیری شود

SUCCESS CRITERIA

اگر یک کاربر جدید بدون آموزش بتواند:

1. جستجو کند
2. کسب‌وکار پیدا کند
3. محصول ببیند
4. ذخیره کند
5. ثبت‌نام کند
6. کسب‌وکار ثبت کند
7. وارد پنل کسب‌وکار شود

در کمتر از 5 دقیقه

UX موفق محسوب می‌شود.
