# آرشیو

پروژه‌های غیرفعال که از workspace اصلی (`pnpm`) خارج شده‌اند.

| پوشه | توضیح | چرا آرشیو شد |
|------|--------|----------------|
| `seller-app/` | اپ Expo/React Native فروشنده | قابلیت‌ها به `artifacts/web` (`/business`) منتقل می‌شود |
| `mockup-sandbox/` | پیش‌نمایش mockupهای طراحی در Replit Canvas | فقط ابزار طراحی؛ بخشی از اپ production نیست |

برای اجرای برنامه فقط به `artifacts/web` و `artifacts/api-server` نیاز است — راهنما: `RUN.md`.
